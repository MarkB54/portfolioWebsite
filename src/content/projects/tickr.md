## Overview

Tickr is a full-stack ticketing application built as part of UNSW coursework. The project demonstrates event-driven architecture patterns for scalable systems. Still working on it.

## Key Achievements

- Deployed a full-stack microservices application using Node.js, Express, React, and Next.js with server-side rendering
- Containerised and orchestrated multi-service application using Docker and Kubernetes
- Implemented JWT-based authentication for secure access control
- Developed comprehensive testing suite and reusable NPM packages to ensure service reliability and code maintainability
